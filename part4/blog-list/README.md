# Blog List Backend

Full Stack Open part 4 blog list backend, expanded through the user and token authentication exercises.

## Setup

1. Use Node.js 22.3.0 or newer.
2. Run `npm install`.
3. Copy `.env.example` to `.env` and set `MONGODB_URI` to a local MongoDB or MongoDB Atlas connection URI.
4. Start development mode with `npm run dev`.

The API is available at `http://localhost:3003/api/blogs`. Use `requests/blogs.rest` with the VS Code REST Client extension to add and retrieve blogs.
